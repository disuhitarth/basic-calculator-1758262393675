## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/basic-calculator.git
```

2. Navigate to project directory
```bash
cd basic-calculator
```

3. Install dependencies
```bash
npm install
```

4. Start development server
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`

## Usage

- Use the numeric keypad or click buttons to input numbers
- Perform calculations using operation buttons (+, -, Ã—, Ã·)
- Press 'C' or click Clear button to reset
- View calculation history in the side panel
- Keyboard shortcuts:
  - Enter/Return: Calculate result
  - Escape: Clear calculator
  - Backspace: Delete last input
  - Numbers 0-9: Input numbers
  - Operators (+, -, *, /): Perform operations
